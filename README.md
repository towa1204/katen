# katen
## ファイル 
**drawLandmark.py**
- webカメラで表示された顔に対して、顔認識を行い取得した顔パーツの座標を表示します。
 
**digitalMakeup.py**
- webカメラで表示された顔を複数認識し、指定したスタンプを貼り付けます。
 
  画像ファイルは透過機能が付いた画像ファイル(pngなど)のみ対応しています。
   
  [digitalMakeup.pyの使い方について](/digitalMakeupDoc.md)
